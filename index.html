My projects

Auction

E-commerce auction site based on Django
That allows users to post auction listings, place bids and comment on listings, and add listings to a watchlist.

Active Listings: displays all of the currently active auction listings. Signed-in user can add them to Watchlist.

Listing: clicking on a listing takes users to the listing page, where users can view all details about the listing. Signed-in user can add comments, add or remove the item to Watchlist as well as bid on the item. Creator of the listing can close the auction from this page.

Create Listing: to create a listing, user must specify a title, description and the starting bid. Users can provide a URL for an image for the listing and/or a category (existing or new one).

Watchlist: displays all the listings that user has added to their watchlist.

Categories: displays a list of all listing categories. Clicking on the any category displays all of the active listings in that category.

Mail

This is an single-page-app email app based on Django and JavaScript
That allows users to send and receive emails, read, answer and archive them.

Inbox: displays all received emails. If the email is read/unread, it appears with a blue/pink background accordingly.

Sent: displays all sent emails.

Archived: displays all archived emails.

Compose: User can send an email to existing user stating subject and body of the email.

Email: Clicking on the email, the user is shown a content of that email. From the inbox page, they can be replied and/or archived. From the archive page, the can be unarchived.

My Games

A web-based application based on Flask and Javascript
That allows users to collect all memories about played games for board games enthusiasts :)

Games: displays all board games in which you played. Games can be sorted by name, rating, date, price or owning. Adding a new game button is also here.

Random: here first player can be chosen or made a random order of players. To do so, press the "start" button, type one letter/number for each person and then press "first player" or "random order". You can press again to change if you want.

More games: displays a list of websites where you can search for new interesting games. At first there will be default sites but you can delete them and add new ones.
